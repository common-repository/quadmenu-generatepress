Create a Mega Menu in GeneratePress.

== Description ==
[Demo](https://quadmenu.com/generatepress/) | [Documentation](https://quadmenu.com/documentation/) | [Developers](https://quadmenu.com/documentation/developers/) | [Community](https://www.facebook.com/groups/quadlayers/)

This WordPress Plugin allow you to create a Mega Menu in GeneratePress and integrates QuadMenu with the GeneratePress theme settings. Requires QuadMenu and GeneratePress.

== Screenshots ==

Demo Site: https://quadmenu.com/generatepress/

1. QuadMenu Mega Menu working with GeneratePress
2. This plugin adds a preconfigured "GeneratePress" theme to fit QuadMenu Mega Menu settings.

== Installation ==

1. Go to the Plugins page in WordPress
1. Search for "GeneratePress Mega Menu"
1. Click "Install"

After installation go to Appearance > Menus and create your mega menu.
